# Design Patterns Quiz Notes

Source: Java Guides Design Patterns Quiz

## 1. What is a design pattern in software development?

**Answer:** A general reusable solution to a commonly occurring problem.

---

## 2. How many types of design patterns are there?

**Answer:** 3

* Creational
* Structural
* Behavioral

---

## 3. What is the main benefit of using a design pattern?

**Answer:** All of the above

* Reduces code complexity
* Improves maintainability
* Separates responsibilities

---

## 4. Which of the following is NOT a creational design pattern?

**Answer:** Bridge Pattern

---

## 5. Which of the following is a behavioral design pattern?

**Answer:** Observer Pattern

---

## 6. What is the Singleton design pattern?

**Answer:** Ensures a class has only one instance and provides a global access point to it.

---

## 7. What is the Factory Pattern?

**Answer:** Creates objects without exposing creation logic and returns them through a common interface.

---

## 8. What is the use of the Builder Pattern?

**Answer:** Simplifies creation of complex objects by constructing them step by step.

---

## 9. What is the purpose of the Abstract Factory Pattern?

**Answer:** Creates families of related or dependent objects without specifying their concrete classes.

---

## 10. What is the purpose of the Prototype Pattern?

**Answer:** Creates new objects by cloning existing objects.

---

## 11. What is the purpose of the Adapter Pattern?

**Answer:** Allows incompatible interfaces to work together.

---

## 12. Which pattern helps reduce complex conditional logic?

**Answer:** Strategy Pattern

---

## 13. What is another name for the Decorator Pattern?

**Answer:** Wrapper

---

## 14. Which structural pattern adds responsibilities dynamically?

**Answer:** Decorator Pattern

---

## 15. What is the Composite Pattern?

**Answer:** Composes objects into tree structures to represent part-whole hierarchies.

---

## 16. Which pattern provides sequential access to elements without exposing implementation details?

**Answer:** Iterator Pattern

---

## 17. Which pattern is concerned with communication between objects?

**Answer:** Mediator Pattern

---

## 18. Which pattern uses a surrogate or placeholder object to control access?

**Answer:** Proxy Pattern

---

## 19. Which pattern decouples abstraction from implementation?

**Answer:** Bridge Pattern

---

## 20. Which pattern allows an object to change behavior when its internal state changes?

**Answer:** State Pattern

---

## 21. Which pattern performs operations on an object structure without changing its classes?

**Answer:** Visitor Pattern

---

## 22. Which pattern defines a family of algorithms and makes them interchangeable?

**Answer:** Strategy Pattern

---

## 23. Which pattern defines a one-to-many dependency where dependents are automatically notified?

**Answer:** Observer Pattern

---

## 24. Which pattern encapsulates a request as an object?

**Answer:** Command Pattern

---

## 25. What is the Template Method Pattern?

**Answer:** Defines the skeleton of an algorithm while allowing subclasses to override specific steps without changing the overall structure.

---
