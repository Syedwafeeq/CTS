# DSA Quiz Notes

This folder contains notes, observations, and quiz-related learning outcomes from the Data Structures and Algorithms (DSA) module of the Cognizant Digital Nurture 5.0 (DN 5.0) DeepSkilling program.

## Topics Covered

### Analysis of Algorithms
- Time Complexity
- Space Complexity
- Big O Notation
- Recursive Complexity Analysis

### Arrays
- Array Traversal
- Searching in Arrays
- Array Operations
- Time Complexity of Array Operations

### Searching Algorithms
- Linear Search
- Binary Search
- Best, Average, and Worst Case Analysis

### Sorting Algorithms
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort

### Linked Lists
- Singly Linked List
- Doubly Linked List
- Circular Singly Linked List
- Circular Doubly Linked List
- Linked List Operations

### Exercises Completed
- E-Commerce Search Function
- Financial Forecasting

## Learning Outcomes

- Understand fundamental data structures and algorithms.
- Analyze algorithm efficiency using time and space complexity.
- Implement searching and sorting techniques.
- Work with different types of linked lists.
- Apply DSA concepts to practical problem-solving scenarios.
- Develop optimized and maintainable solutions.
