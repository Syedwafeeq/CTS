# Recursive Complexity

Recursion occurs when a function calls itself.

## Examples

- Factorial
- Fibonacci
- Merge Sort
- Quick Sort

## Complexity Examples

- Factorial: O(n)
- Fibonacci (naive): O(2^n)
- Merge Sort: O(n log n)
- Quick Sort: O(n log n) average

Recursive algorithms use additional stack space during execution.
