# JavaScript

This folder contains self-practice exercises created as part of the Cognizant Digital Nurture 5.0 (DN 5.0) Upskilling program.

## Files

1. variables_demo.html
   - Demonstrates variables and data types.

2. operators_demo.html
   - Demonstrates operators and conditional statements.

3. loops_demo.html
   - Demonstrates for loop, while loop, break, and continue.

4. functions_demo.html
   - Demonstrates functions, parameters, return values, and arrow functions.

5. arrays_objects_demo.html
   - Demonstrates arrays, objects, and array methods.

6. dom_demo.html
   - Demonstrates DOM manipulation and event handling.

7. es6_demo.html
   - Demonstrates ES6 features such as destructuring and spread operator.

8. async_await_demo.html
   - Demonstrates asynchronous programming using async and await.

9. fetch_api_demo.html
   - Demonstrates API calls using Fetch API.

## Learning Outcomes

- Understand JavaScript fundamentals
- Work with variables and operators
- Implement loops and functions
- Manipulate arrays and objects
- Modify the DOM dynamically
- Use modern ES6 features
- Handle asynchronous operations
- Consume APIs using Fetch
