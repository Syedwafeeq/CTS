# ADO.NET

This folder contains self-practice exercises created as part of the Cognizant Digital Nurture 5.0 (DN 5.0) Upskilling program.

## Files

1. ConnectionDemo.cs
   - Demonstrates establishing and closing database connections.

2. InsertData.cs
   - Demonstrates inserting records using parameterized queries.

3. ReadData.cs
   - Demonstrates retrieving records using SqlDataReader.

4. UpdateDelete.cs
   - Demonstrates updating and deleting records.

## Learning Outcomes

- Connect C# applications with SQL databases
- Execute parameterized SQL queries
- Read data using SqlDataReader
- Perform CRUD operations
- Apply exception handling in database applications
