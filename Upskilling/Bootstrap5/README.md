# Bootstrap5

This folder contains self-practice exercises created as part of the Cognizant Digital Nurture 5.0 (DN 5.0) Upskilling program.

## Files

1. grid_system.html
   - Demonstrates Bootstrap containers, rows, columns, and responsive layouts.

2. navbar_demo.html
   - Demonstrates responsive navigation bars and Bootstrap components.

3. cards_demo.html
   - Demonstrates cards, buttons, shadows, and utility classes.

4. forms_demo.html
   - Demonstrates Bootstrap forms, input groups, and form controls.

## Learning Outcomes

- Understand Bootstrap grid systems
- Create responsive layouts
- Build navigation bars and cards
- Design professional forms
- Apply Bootstrap utility classes
