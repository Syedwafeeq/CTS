# HTML5

This folder contains self-practice exercises created as part of the Cognizant Digital Nurture 5.0 (DN 5.0) Upskilling program.

## Files

1. basic_structure.html
   - Demonstrates HTML5 document structure and semantic tags.

2. registration_form.html
   - Demonstrates HTML forms and input elements.

3. student_table.html
   - Demonstrates tables and lists.

4. navigation_demo.html
   - Demonstrates navigation using hyperlinks and section references.

5. local_storage_demo.html
   - Demonstrates HTML5 Local Storage API.

6. geolocation_demo.html
   - Demonstrates HTML5 Geolocation API.

## Learning Outcomes

- Understand HTML5 semantic elements
- Create forms and tables
- Implement navigation within web pages
- Use Local Storage for persistent data
- Access browser geolocation services
