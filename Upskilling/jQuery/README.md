# jQuery

This folder contains self-practice exercises created as part of the Cognizant Digital Nurture 5.0 (DN 5.0) Upskilling program.

## Files

1. selectors_demo.html
   - Demonstrates jQuery selectors, events, and text manipulation.

2. effects_demo.html
   - Demonstrates hide, show, toggle, fadeIn, and fadeOut effects.

3. dom_manipulation.html
   - Demonstrates dynamic DOM manipulation using html, append, prepend, and css methods.

## Learning Outcomes

- Understand jQuery selectors
- Handle events using jQuery
- Apply visual effects and animations
- Manipulate DOM elements dynamically
- Modify styles programmatically
