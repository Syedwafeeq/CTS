# CSS3

This folder contains self-practice exercises created as part of the Cognizant Digital Nurture 5.0 (DN 5.0) Upskilling program.

## Files

1. selectors_demo.html
   - Demonstrates CSS selectors including element, class, ID, and inline styling.

2. box_model_demo.html
   - Demonstrates margin, padding, border, outline, visibility, and display properties.

3. responsive_design.html
   - Demonstrates responsive web design using media queries.

4. tables_and_links.html
   - Demonstrates styling of tables, hyperlinks, lists, colors, and fonts.

5. columns_demo.html
   - Demonstrates CSS multiple column layouts.

## Learning Outcomes

- Understand CSS syntax and selectors
- Apply colors, fonts, and text styling
- Design responsive web pages
- Work with the CSS box model
- Style tables, links, and lists
- Implement multiple column layouts
