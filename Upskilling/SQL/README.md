# SQL

This folder contains self-practice SQL exercises created as part of the Cognizant Digital Nurture 5.0 (DN 5.0) Upskilling program.

## Files

1. create_tables.sql
   - Demonstrates database and table creation with constraints.

2. insert_data.sql
   - Demonstrates insertion of sample data into tables.

3. select_queries.sql
   - Demonstrates SELECT, WHERE, DISTINCT, ORDER BY, and LIMIT.

4. joins.sql
   - Demonstrates INNER JOIN and LEFT JOIN operations.

5. aggregate_functions.sql
   - Demonstrates COUNT, AVG, MAX, MIN, SUM, GROUP BY, and HAVING.

6. subqueries.sql
   - Demonstrates nested queries and subqueries.

7. indexes_constraints.sql
   - Demonstrates indexes and additional constraints.

## Learning Outcomes

- Design relational databases
- Create and manage tables
- Retrieve and filter data
- Perform joins between tables
- Use aggregate functions
- Write subqueries
- Apply indexes and constraints
